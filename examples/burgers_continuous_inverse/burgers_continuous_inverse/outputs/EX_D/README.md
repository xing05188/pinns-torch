# 实验 D：网络容量消融

## 配置
- **配置文件**: configD.yaml
- **网络结构**: [2, 10, 10, 10, 10, 1]（小网络）
- **N_u**: 2000
- **max_epochs**: 40000
- **seed**: 1234

## 结果
| 指标 | 值 | 真实值 | 偏差 |
|------|-----|--------|------|
| val/error_u | 0.0228 | - | - |
| val/loss | 0.000196 | - | - |
| l1 | 0.9795 | 1.0 | ~2% |
| l2 | -5.477 | -6.0 | ~8.7% |

## 分析
小网络导致 `l2` 参数识别偏差较明显（约 8.7%），而 `l1` 偏差较小（约 2%）。这表明网络容量对参数辨识有一定影响。
